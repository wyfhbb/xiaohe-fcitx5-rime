# Flypy Homophones Lexicon for Fcitx5-Rime

Personal homophone dictionary for Fcitx5-Rime with Flypy schema on Linux.

## Installation

Extract to `~/.local/share/`:
```bash
tar -xzf flypy.tar.gz -C ~/.local/share/
```

This will overwrite the `fcitx5` directory with the configured lexicon.

## Contents

- Optimized homophones database for Flypy input
- Fcitx5-Rime configuration files

## Requirements

- Fcitx5
- Rime engine for Fcitx5
- Linux environment
